HTML stands for HyperText Markup Language. It a markup language used in web development, to create web page and application. By web pages, we mean the individual documents on the world wide web(WWW) that can be accessed throug a web browser, such as chrome, opera mini, fire fox microsoft edge etc. They are created using various web development technologies such as HTML, CSS, JavaScript etc and are hosted on web server.

HOW TO VIEW AN HTML SOURCE:

Right-click in an HTML page and select VIEW SOURCE CODE (in chrome) or VIEW SOURCE CODE (in edge) or similar in other browsers. this will open a window containing source code of the page.

HOW TO INSPECT AN HTML ELEMENT

Right-click on an element (or on a blank area) and choose INSPECT or INSPECT ELEMENT to see what elements are made up of (you will see HTML and the CSS). You can also edit the HTML and CSS on-the-fly in the element or style panel that opens.

HTML Style-CSS:

CSS stands for Cascading Style Sheets. CSS serves a lot of work. It can control the layout of multiple web pages all at once.
Cascading style sheet (CSS) is ued to format the layout of a web page. With CSS, you can control the color,font, the size of the text, the spacing between the elements, how elements positioned and laid out, what background images or background colors are to be used, different display for different devices and screen sizes and more.
The word cascading means that a style applied to a parent will also apply to all the children elements within the parent. So if you set the color of the body to BLUE, all headings, paragraph and other text element within the body will also get the same color (unless you specify something else).
